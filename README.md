# BJK University — Webinar Analytics Dashboard

A production-grade, interactive analytics dashboard built with Plotly Dash for BJK University's Wednesday webinar: *"A 30-Day Path to Becoming an AI Business Strategist Businesses Actually Pay For"* (April 1, 2026). The dashboard provides a complete funnel breakdown—registered → attended → booked calls—split by **Organic vs. Paid** traffic sources, with engagement analysis, registration velocity, and a filterable data table. Built to replace spreadsheet reporting with a live, shareable URL.

## Quick Start

```bash
pip install -r requirements.txt
python app.py
# Open http://localhost:8050
```

## Deployment

Configured for Render.com free tier. Push to GitHub and connect via `render.yaml`.
